# 🛒 Local Store Management System

A web-based store management system built to help local businesses 
manage their products, stock, and sales efficiently.

## 🚀 Features
- Add, update, and delete products
- Track product quantity in real time
- View total sales and revenue summary
- Simple and clean dashboard UI

## 🛠️ Tech Stack
- **Frontend:** HTML, CSS, JavaScript
- **Database:** MongoDB
- **Editor:** Cursor (AI-assisted development)
